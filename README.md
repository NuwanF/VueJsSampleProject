# VueJsSampleProject

# Set-up

## install vue
npm install vue

## cli
npm install -g @vue/cli

## create project
vue create test-project
cd test-project

## add component library
vue add vuetify

## sass loader and node-sass is required to working with vuetify
npm install sass-loader node-sass webpack --save-dev

## add vue router
npm  install vue-router

## http client
npm i vue-axios
npm i axios
